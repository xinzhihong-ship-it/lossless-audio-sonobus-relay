## This Modified Project

This repository modifies SonoBus and adds:

- Self-hosted connection server deployment.
- Public relay support for clients behind NAT or without public IP addresses.
- Web administration for viewing connections, kicking users, banning users, and unbanning users.
- Docker-based Linux server deployment.
- Documentation and build workflows for Windows, macOS, and Linux artifacts.
